## 🎓 Description

UX Tools MVP is an academic web platform designed to teach:

-   Card Sorting (Open / Closed / Hybrid)
-   Tree Testing (Treejack-style)

Built for lightweight shared hosting environments:

-   PHP 8.x (framework ligero)
-   MySQL
-   Vanilla JavaScript
-   Tailwind CSS (CDN)

------------------------------------------------------------------------

## 🎯 Academic Purpose

The platform enables:

-   Simulation of real UX research workflows
-   Team-based collaborative configuration
-   Execution of studies with external participants
-   Analysis of consensus and navigation success metrics

------------------------------------------------------------------------

## 🧠 Research Context

This system serves as:

-   A controlled experimental platform in educational UX research
-   A lightweight HCILab environment
-   A teaching instrument for methodological experimentation
